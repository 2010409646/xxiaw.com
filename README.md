# 介绍

`下下软件系统` 是为 `小白` 用户提供一个0开发成本做一个软件下载站，来开发项目的。

软件分为软件模块以及资讯模块，可以更好的让你进行傻瓜式操作

[官方说明](https://www.xxiaw.com/cms/guide/background.html)  (PC版本)


##环境要求

因系统采用THINKPHP3.2内核开发

[环境要求如下]

php版本要求：>php5.3 <7.0

mysql：5.5以上

web服务器：nginx 、IIS、Apache
